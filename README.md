# Blur.

Blur is **heavily** inspired by [motionblur](https://modrinth.com/mod/motionblur), but with a proper *Config GUI* and *Smart Blur™* which is a fancy way of saying it disables itself under x fps so your game looks less bad on low fps.

## Dependencies:
- [oωo (owo-lib)](https://modrinth.com/mod/owo-lib)
- [Mod Menu](https://modrinth.com/mod/modmenu)
- [Satin API](https://modrinth.com/mod/satin-api)
